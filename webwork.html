KickBack! is a web application that uses third-party APIs to generate randomized movie and recipe suggestions based on the user’s input of genre and cuisine type. The motivation for the project was to make it easier for people to stay in and spend quality time with their loved ones during the pandemic.

I co-created this concept with three other classmates in my UCLA Coding Bootcamp cohort. We developed the design and code over a 10 day period. I was primarily focused on UI/UX and front-end development.

Tech: HTML5 / CSS3 / JavaScript / APIs

ZooKeepr is a web application to keep track of a fictional zoo's animals and zookeepers. Built with a custom server-side API using the Express.js framework.

Capabilites include:
1. User can add a zookeeper and/or animal with specific traits to the database.

2. Zookeepers and animals added to database can be searched according to unique IDs.

Tech: HTML5 / CSS3 / JavaScript / Node / Express / Jest / Insomnia / Heroku

Note Taker is a web application to keep track of all of life's "to-dos." Built with a custom server-side API using the Express.js framework and deployed with Heroku.

Capabilites include:
1. User can add title and information to each note.

2. Notes can be saved and deleted.

3. After notes are saved, they persist in the custom database.

Tech: HTML5 / CSS3 / JavaScript / Node / Express / Heroku
