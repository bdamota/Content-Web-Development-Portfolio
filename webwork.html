Have to stay in? Make a night of it.

KickBack! is a web application that uses third-party APIs to generate randomized movie and recipe suggestions based on the user’s input of genre and cuisine type. The motivation for the project was to make it easier for people to stay in and spend quality time with their loved ones during the pandemic.

I co-created this concept with three other classmates in my UCLA Coding Bootcamp cohort. We developed the design and code over a 10 day period. I was primarily focused on UI/UX and front-end.

Tech: HTML5 / CSS3 / JavaScript / APIs

A simple dashboard app to find out the weather conditions in multiple cities. The app calls on the OpenWeather API for information.

When the user searches for a city, they are presented with the following:

1. Current weather conditions, including temp, humidity, wind speed, UV Index.

2. The 5-day forecast.

After the initial search, the city is added to the historical search.

Tech: HTML5 / CSS3 / JavaScript / JQuery / API
